# Demo GIF

Record your screen showing: paste PRD → `/prd-to-kanban` fires → `work/kanban.md` appears.

## How to record

**Recommended tools:**
- [Kap](https://getkap.co) — open-source, native macOS, exports GIF/WebM
- [LICEcap](http://www.cockos.com/licecap) — simple, Windows/macOS

**Steps:**
1. Start screen recording covering Claude Code window
2. Paste a sample PRD
3. Run `/prd-to-kanban`
4. Show the generated `work/kanban.md` content
5. Stop recording → save as `demo.gif` in this directory

**Tip:** Keep it under 15 seconds, show the key moment (PRD paste → board generation), loop-friendly format.
